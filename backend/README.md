# Backend

This directory contains the prototype backend implementation of the Business Operations AI Platform.

## Current Features

- Employee API
- Shift API
- Daily Reporting API
- Analytics Summary API
- Health Check Endpoint

## Available Endpoints

### GET /

Returns API status information.

### GET /health

Health check endpoint.

### GET /employees

Returns employee records.

### GET /shifts

Returns shift records.

### GET /reports/daily

Returns a sample daily operations report.

### GET /analytics/summary

Returns sample workforce analytics.
